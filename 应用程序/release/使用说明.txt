AI 辩论场 · 独立发行版 使用说明
================================

【启动】
  双击 AI辩论场.exe 即可。无需安装 Python / Node，无需 start.bat。

【配置】
  打包版若已含 .env 则可直接使用；否则将 .env.example 另存为 .env 并填入 API 密钥。
  菜单：文件 → 打开配置目录（编辑 .env）

【局域网联机】
  本版已内置联机（后端与页面均监听局域网）。
  主持：创建联机房间 → 菜单「复制联机地址（局域网）」→ 发给同学
  同学：浏览器打开该地址，选席位加入（无需安装本程序）
  防火墙需放行端口 5173（页面）与 9000（API）。

【公网联机（内网穿透）】
  首页或联机大厅 → 「开启公网穿透」→ 复制 https://…trycloudflare.com 地址
  创建房间后，加入链接会自动使用公网地址（远程同学无需同一 Wi‑Fi）

【目录说明】
  AI辩论场.exe          主程序
  .env / .env.example   API 与运行配置（与 exe 同目录）
  resources/app-core/   内置 Python、后端、前端资源（勿删）
  其他 dll / pak 文件   Electron 运行依赖

【故障排除】
  · 端口占用：关闭其他辩论程序窗口后重试
  · 无法联机：确认同一 Wi‑Fi 或已开启公网穿透
  · 白屏/启动超时：确认 9000、5173 未被占用

【数据】
  辩论数据默认在内存模式（未连接 MongoDB 时）。
  向量索引等文件位于 resources/app-core/data/
